วีไอ และวีไอเอ็ม (Vi and Vim)
Vi คืออะไร
Vi (Visual Editor) คือ Text Editor หรือเรียกสั้น ๆ ว่า Editor ใช้สำหรับ เพิ่ม ลบ แก้ไข อักษรในไฟล์ข้อมูลที่เป็นประเภท Text File

โหมดการทำงานของโปรแกรมวีไอ
- Command Mode คือ โหมดที่สามารถใช้คำสั่งได้
- Insert Mode คือ โหมดที่พิมพ์ข้อมูลตัวอักษรลงไป

วิธีเรียกใช้งานโปรแกรม Vi ในระบบปฏิบัติการ Unix
ให้พิมพ์คำสั่ง vi แล้วกดปุ่ม Enter วิธีเปิด Text File มาแก้ไข โดยใช้โปรแกรม Vi
ให้พิมพ์คำสั่ง vi ตามด้วยชื่อไฟล์.นามสกุลไฟล์
ยกตัวอย่าง เช่น vi file_name.txt

วิธีกู้คืนไฟล์ ขณะที่กำลังแก้ไขแล้วเกิดเหตุขัดข้อง
ให้พิมพ์คำสั่ง vi -r ชื่อไฟล์.นามสกุลไฟล์

Vim คืออะไร
Vim (Vi Improved) คือ Text Editor ที่พัฒนาต่อมาจาก Vi ใช้สำหรับ เพิ่ม ลบ แก้ไข อักษรในไฟล์ข้อมูลที่เป็นประเภท Text File เหมือนกัน แต่จะเพิ่มความสามารถบางอย่างเข้าไป เช่น Syntax Highlight เป็นต้น

โหมดการทำงานของโปรแกรม Vim
Normal Mode หรือ Command Mode คือ โหมดที่สามารถใช้คำสั่งคีย์ลัดและพิมพ์คำสั่งได้
Insert Mode คือ โหมดที่เราสามารถพิมพ์ข้อความลงไปได้
Visual Mode คือ โหมดที่สามารถนำเมาส์ลากคลุมดำได้

วิธีติดตั้ง Vim ในระบบปฏิบัติการ Linux
ให้พิมพ์คำสั่ง sudo apt-get install vim ใน Terminal

วิธีสร้างหรือเปิด Text File มาแก้ไข โดยใช้โปรแกรม Vi
ให้พิมพ์ vim ชื่อไฟล์.นามสกุลไฟล์
ยกตัวอย่าง เช่น vim file_name.txt

วิธีเข้าสู่ Insert Mode เพื่อแก้ไขหรือพิมพ์ข้อความ
เมื่อเข้าสู่โปรแกรม Vim แล้ว ก็ให้กดปุ่ม insert หรือ i แล้วพิมพ์สิ่งที่ต้องการได้เลย
เราสามารถออกจาก Insert Mode เพื่อกลับสู่ Normal Mode ได้โดยการกดปุ่ม Esc ที่แป้นพิมพ์

แสดงเลขบรรทัดในโปรแกรม Vim
ให้ท่านอยู่ใน Normal Mode
ให้กดปุ่ม Esc ที่แป้นพิมพ์ :set number แล้วกดปุ่ม Enter ที่แป้นพิมพ์

วิธีทำให้สามารถพิมพ์ภาษาไทยในโปรแกรม Vim ได้
ให้ดาวน์โหลด VIM-THAI-KEYS จาก https://github.com/chakrit/vim-thai-keys
หากท่านใช้ Windows ให้นำไฟล์ vim-thai-keys.vim ใส่ไว้ใน C:\Program Files (x86)\Vim\vim80\keymap\vim-thai-keys.vim (หรือใส่ไว้ที่โฟลเดอร์ keymap ตำแหน่งที่ท่านได้ติดตั้งโปรแกรม Vim เอาไว้)
เข้าโปรแกรม Vim แล้วพิมพ์ set keymap=vim-thai-keys แล้วกดปุ่ม Enter ที่แป้นพิมพ์

วิธีกำหนด Character Encoding ให้กับ Vim
ให้ท่านอยู่ใน Normal Mode ก่อน
- เปลี่ยนการแสดงผลบนหน้าจอเป็น UTF-8 ให้กดปุ่ม Esc ที่แป้นพิมพ์ พิมพ์ :set encoding=utf-8 แล้วกด enter
- เปลี่ยนไฟล์เป็นการเข้ารหัสแบบ UTF-8 ให้กดปุ่ม Esc ที่แป้นพิมพ์ พิมพ์ :set fileencoding=utf-8 แล้วกด enter

การออกจากโปรแกรม
การออกจากโปรแกรมท่านต้องอยู่ใน Normal Mode ก่อน หากท่านอยู่ใน Insert Mode หรือ Visual Mode ให้ท่านกด Esc กลับสู่ Normal Mode ก่อน
- การออกจากโปรแกรม ให้กดปุ่ม Esc ที่แป้นพิมพ์ แล้วพิมพ์ :q แล้วกดปุ่ม Enter ที่แป้นพิมพ์
- การออกจากโปรแกรมโดยไม่บันทึก ให้กดปุ่ม Esc ที่แป้นพิมพ์ แล้วพิมพ์ :q! แล้วกดปุ่ม Enter ที่แป้นพิมพ์
- การออกจากโปรแกรมและบันทึกข้อมูลด้วย ให้กดปุ่ม Esc ที่แป้นพิมพ์ แล้วพิมพ์ :wq แล้วกดปุ่ม Enter ที่แป้นพิมพ์
